Teacher Grade Analysis App

Compare National Test results with yearly performance, final grades, questions, topics, gender and class sections.

1. Students

Add students manually or upload a CSV/Excel list. Student list format: Name, Gender, Class/Section. Gender and class are optional.

2. National Test Import

Upload your National Test CSV. The app understands the format where row 1 contains max points, row 2 contains Name, optional gender, question numbers, and later columns include Total, NT Grade, Teacher Grade, Final Grade and deviation notes.

3. Rest of the Year / Class Assessments

Optional: upload class assessment data to compare the National Test with yearly performance. Format: Name, Assessment, Score, Max, Grade, Topic, Subject, Date. Only Name, Score and Max are required.

Add one class assessment manually

4. Question Topics, Question Type and Skills

After importing an NT file, upload the question mapping CSV. Format: Part, Question, Max Points, Topic, Question Type, Skill / Ability, Knowledge Area, E/C/A Level, Notes / What students needed to show.
